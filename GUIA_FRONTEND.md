# Configuração do Frontend na Vercel

O seu frontend (pasta `dands-cadastro-usuarios`) ainda não está conectado ao GitHub. Siga estes passos:

1.  Crie um novo repositório no GitHub chamado `frontend-cadastro`.
2.  No terminal (dentro da pasta `dands-cadastro-usuarios`), execute:
    ```bash
    git remote add origin https://github.com/SEU_USUARIO/frontend-cadastro.git
    git branch -M main
    git push -u origin main
    ```
3.  Vá na Vercel e importe esse projeto.
